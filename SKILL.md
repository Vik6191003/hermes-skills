# Hermes Skills Repo

This repo holds reusable skills for the Hermes AI agent running for Alexander Barnes (Vik61).

Skills here are auto-discovered and loaded by Hermes when relevant tasks arise.

## How to add a skill
1. Create a folder: `skill-name/`
2. Add `SKILL.md` with trigger conditions, steps, and pitfalls
3. Push to GitHub — Hermes auto-loads on next relevant task

## Categories
- `code-review` — Code review workflows
- `research` — Web research and data gathering  
- `gaming` — Xbox and game automation
- `deploy` — Deployment and DevOps
