# OpenViking — Context Database for AI Agents

## What It Does
OpenViking unifies context management (memory, resources, skills) through a filesystem paradigm:

```
/context/
  /memory/       — persistent long-term memory
  /resources/     — files, URLs, data sources
  /skills/        — executable capabilities
```

## Why Use It
- Better context delivery across sessions
- Self-evolving knowledge base
- Hierarchical context organization
