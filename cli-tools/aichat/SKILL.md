# aichat CLI Integration

## Commands
```bash
aichat --help                   # Show all commands
aichat -m claude "prompt"      # Use specific model
aichat --shell                  # Shell assistant mode
```

## Supported Models
OpenAI, Claude, Gemini, Ollama, Groq, Local LLMs

## How I Use It
Quick model comparison and shell assistance. Also useful for multi-model testing.
