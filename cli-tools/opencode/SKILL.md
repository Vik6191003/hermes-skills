# OpenCode CLI Integration

## Commands
```bash
opencode --version              # Check version
opencode --help                 # Show all commands
opencode agents list            # List available agents
opencode run <cmd>             # Run a command
```

## How I Use It
OpenCode wraps websites/APIs as CLI commands. I use it to integrate external services into my toolset.

## Config Location
~/.opencode/config.yaml
