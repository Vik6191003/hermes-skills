# Web Research Skill

## Trigger
User asks for: "research", "find out", "look into", "what's the latest on", "compare X vs Y"

## Steps
1. Identify 3-5 specific search queries to cover the topic
2. Run parallel web searches using the research tool
3. Synthesize findings — extract key facts, dates, claims
4. For comparison requests: build a clear comparison table
5. Flag claims that need verification with ⚠️
6. End with a clear verdict or recommendation

## Output Format
```
## 📊 Research: [Topic]

### Key Facts
- ...

### Comparison (if applicable)
| Criteria | Option A | Option B |
|---|---|---|
| ...

### ⚠️ Unverified Claims
- ...

### Verdict
[Clear answer to the user's question]
```
